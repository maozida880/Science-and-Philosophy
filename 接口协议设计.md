
# 权益优惠券推荐接口协议

Hbase输入格式：{user_id: "user_id1", model_1: {...}, model_2: {...}}

## 请求参数

| 参数名   | 类型    | 是否必须 | 描述                                           |
|----------|--------|----------|------------------------------------------------|
| user_id  | String | 是       | 用户唯一标识                                   |
| model_id  | String | 是       | 返回模型对于不同方案的得分                      |


## 返回数据

{user_id: "user_id1", model_1: {...}, model_2: {...}}

### 样例

{
 user_id: user_id1, 
 model_1: {
         plan_no_1:plan_no_1,
         score_1:score_1, 
         plan_no_2:plan_no_2,
         score_2:score_2,
         ..., 
         plan_no_10:plan_no_10, 
         score_10:score_10
         }, 
 model_2:{} 
 }
