
# 权益优惠券推荐接口协议

Hbase输入格式：{user_id: "user_id1", model_1: [{}...{}], model_2: [{}...{}]}

## 请求参数

| 参数名   | 类型    | 是否必须 | 描述                                           |
|----------|--------|----------|------------------------------------------------|
| user_id  | String | 是       | 用户唯一标识                                   |
| model_id  | String | 是       | 返回模型对于不同方案的得分                      |


## 返回数据

{user_id: "user_id1", model_1: {...}, model_2: {...}}

### 样例

{
 user_id: user_id1, 
 model_1:   [
      {
        "plan_no": "PLAN_A001",
        "score": 0.95
      },
      {
        "plan_no": "PLAN_B042",
        "score": 0.87
      },
      {
        "plan_no": "PLAN_C103",
        "score": 0.76
      },
      ...,
      {
        "plan_no": "PLAN_J110",
        "score": 0.99
      }
    ],
         
 model_2:[
      {
        "plan_no": "PLAN_A001",

        "plan_no": "PLAN_B042",

      }
    ]
 }
